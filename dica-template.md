Descreva a dica. Utilize listas, **negrito**, *italico*, e o que for preciso para passar a mensagem.

Tente usar linguagem clara e prática.

```html
<div class="dica"></div>
```

```css
.dica {
  background-color: #fff;
  border: 1px solid #000;
  border-radius: 4px;
}
```

```js
console.log(
  "Se o seu script não precisar de JS, pode excluir esse bloco."
)
```
